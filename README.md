# 📇 Mi Agenda - Desktop Application

Una aplicación de escritorio moderna y elegante para la gestión de contactos y tareas, desarrollada en **C#** utilizando **Windows Forms**. El proyecto se centra en ofrecer una experiencia de usuario fluida con una interfaz limpia basada en el color Teal y una navegación intuitiva.

## ✨ Características

- **Gestión de Contactos:** Permite agregar, editar y eliminar contactos con una interfaz de tarjetas visuales.
- **Categorización Dinámica:** Organización de contactos por categorías personalizables (Trabajo, Familia, Personal, etc.).
- **Favoritos:** Sistema de marcado rápido para acceder a tus contactos más importantes con un solo clic.
- **To-Do List Integrada:** Un módulo adicional para gestionar tareas pendientes dentro de la misma aplicación.
- **Interfaz Moderna:** - Sidebar dinámico con alineación perfecta de iconos.
  - Uso de **FontAwesome** para una iconografía profesional.
  - Tooltips y estados visuales (Active Indicator) para una mejor navegación.
- **Seguridad:** Sistema de autenticación de usuarios (Login/Registro) con validación de credenciales.

## 🚀 Tecnologías Utilizadas

- **Lenguaje:** [C#](https://docs.microsoft.com/en-us/dotnet/csharp/)
- **Framework:** .NET Framework / .NET Core (Windows Forms)
- **Iconografía:** [FontAwesome.Sharp](https://github.com/awesome-inc/FontAwesome.Sharp)
- **Base de Datos:** SQL Server (usando Servicios y Modelos para desacoplamiento de datos)
- **Diseño:** Custom UI con controles personalizados y manejo de eventos de diseño avanzado.
